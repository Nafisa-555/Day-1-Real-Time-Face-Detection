# Real-Time Face Detection using OpenCV

This is a simple Computer Vision project built using Python and OpenCV.  
The program uses the webcam to detect faces in real time and draws a rectangle around detected faces.

## Features
- Real-time webcam face detection
- Uses OpenCV Haar Cascade classifier
- Detects faces from camera and even images shown on screen

## Technologies Used
- Python
- OpenCV

## How It Works
The program captures video from the webcam and converts each frame into grayscale.  
Then it uses a Haar Cascade classifier to detect faces and draws rectangles around them in real time.

## How to Run

1. Install OpenCV
2. Run the program
3. Press **Q** to close the webcam window.
